TaskLoop Icon Generator

Generate icons for the TaskLoop Chrome extension. Click "Generate Icons" to create all required sizes.

Instructions:

  1. Click "Generate Icons" to create the icon files
  2. Right-click each icon and "Save image as..."
  3. Save them in the icons/ folder with the correct names:
    • icon16.png
    • icon32.png
    • icon48.png
    • icon128.png
